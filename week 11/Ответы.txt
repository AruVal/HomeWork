1. Что такое DOM?
Объектная модель документа или DOM - это внутреннее представление веб-страницы в браузере.
Браузер создает модель DOM страницы в процессе загрузки и разбора разметки HTML.

2. Что такое BOM?
Объектная модель браузера (Browser Object Model, BOM) – это дополнительные объекты,
предоставляемые браузером (окружением), чтобы работать со всем, кроме документа.

3. Каким методом можно получить элемент по его идентификатору в DOM?
Если у элемента есть атрибут id, то мы можем получить его вызовом document.getElementById(id), где бы он ни находился.

4. Каким методом можно создать новый элемент в DOM?
DOM-узел можно создать двумя методами:
1) document.createElement(tag);
2) document.createTextNode(text);
Первый создает новый элемент с заданным тегом, второй новый текстовый узел с заданным текстом.

5. Каким методом можно добавить класс к элементу в DOM?
elem.classList.add/remove("class") – добавить/удалить класс.
elem.classList.toggle("class") – добавить класс, если его нет, иначе удалить.

6. Каким методом можно изменить текстовое содержимое элемента в DOM?
С помощью свойства textContent, которое позволяет прочитывать текст этих элементов.

7. Что такое объект события и какие у него могут быть свойства?
Событие – это сигнал от браузера о том, что что-то произошло. Объект события содержит ценную информацию о деталях события.
Он передаётся первым аргументом event в обработчик для всех браузеров, кроме IE8. 
С помощью его свойств мы можем получить такую информацию, как:
type: указывает на имя события;
bubbles: возвращает true, если событие является восходящим;
cancelable: возвращает true, если можно отменить стандартную обработку события;
currentTarget: определяет элемент, к которому прикреплен обработчик события;
defaultPrevented: возвращает true, если был вызван у объекта Event метод preventDefault();
eventPhase: определяет стадию обработки события;
target: указывает на элемент, на котором было вызвано событие;
timeStamp: хранит время возникновения события;

8. Есть элемент: <input id= "input" value = "Привет">. Какой вызов поменяет значение в нём?
Нужно использовать getElementById и затем поменять значение при момощи свойства value.

9. Для чего используется запись: setAttribute(name, value)?
Для добавления нового аттрибута и его значения в уже существующем элементе.

10. Как сделать переход на другую страницу при клике на кнопку (без <a href=...>, только средствами JavaScript)?
Нужно использовать getElementById или querySelector для поиска необходимого элемента, а затем использовать созданную
переменную с атрибутом href для записи новой ссылки.
